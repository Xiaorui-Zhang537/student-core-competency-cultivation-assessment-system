# 全站标题/子标题文案规范（温暖鼓励风）

适用范围：所有使用 `PageHeader` / `PageScaffold` 渲染的页面标题下方“一句话子标题”。

目标：更有性格、更有设计感，同时不油腻、不空泛，读起来像一个可靠的学习伙伴。

## 写作原则

- 一句话：尽量 12-22 字（中文），避免多句堆叠。
- 有动作感：让人知道“下一步做什么”或“现在处于什么状态”。
- 不命令用户：少用“请/必须/管理你的…”，多用“我们一起/慢慢来/稳稳推进”。
- 不夸张：避免“立刻逆袭/彻底改变人生”这类过强承诺。
- 贴合场景：学习类偏陪伴推进；分析类偏趋势与下一步；社区类偏连接与共创；帮助类偏兜底与安心。

## 推荐句式模板（可复用）

学习推进：

- 慢慢来，把今天这 10 分钟学扎实就很好。
- 稳稳推进中，已完成 {progress}%。

任务提醒：

- 把该交的都交上，把该学的都学会。

数据洞察：

- 用数据看见你的进步，下一步更稳。
- 先看趋势，再做选择，我们一起把方向定清楚。

社区氛围：

- 一起讨论、一起进步，把灵感留在这里。

帮助兜底：

- 卡住的时候别硬扛，我们一起把问题解决掉。

## i18n 落点与约定

- 所有 subtitle 必须写入 i18n：
  - 中文：`frontend/src/locales/zh-CN/*.json`
  - 英文：`frontend/src/locales/en-US/*.json`
- 推荐命名：与页面 title 同命名空间，例如：
  - `student.courses.subtitle`
  - `shared.community.subtitle`
  - `teacher.analytics.subtitle`
- 需要插值（例如进度）时使用占位符，例如：
  - `student.lesson.subtitleProgress`: `稳稳推进中，已完成 {progress}%。`

## 反例（尽量避免）

- 管理您的个人信息和账户设置（太像后台系统提示语）
- 发现优质课程，开启学习之旅（太常见、缺少个性）
- 总览您的学习进度和任务（信息正确但不够有温度）
