# 后端 API 参考（按控制器）

本节列出各控制器的端点导航，详细内容在各子页。

- 通用约定（认证/响应/分页/时间）：`/backend/api/conventions`
- 认证（Auth）：`/backend/api/auth`
- 用户资料（Users）：`/backend/api/user`
- 课程（Courses）：`/backend/api/course`
- 章节（Chapter）：`/backend/api/chapter`
- 课时（Lesson）：`/backend/api/lesson`
- 作业（Assignments）：`/backend/api/assignment`
- 提交（Submissions）：`/backend/api/submission`
- 评分（Grades）：`/backend/api/grade`
- 能力评估（Ability）：`/backend/api/ability`
- 学生（Students）：`/backend/api/student`
- 教师（Teachers）：`/backend/api/teacher`
- 社区（Community）：`/backend/api/community`
- 通知（Notifications）：`/backend/api/notification`
- 文件（Files）：`/backend/api/file`
- 帮助中心（Help Center）：`/backend/api/help`
- 聊天（Chat）：`/backend/api/chat`
- 管理员（Admin）：`/backend/api/admin`
- AI 聊天（AI）：`/backend/api/ai`
- AI 批改（AI Grading）：`/backend/api/ai-grading`
- AI 实时语音（Gemini Live）：`/backend/api/ai-live`
- AI 口语训练：`/backend/api/ai-voice-practice`
- 行为证据评价：`/backend/api/behavior`

> 端点详情请以 Swagger 为准：`http://localhost:8080/api/swagger-ui.html`
