# 教师端-学生管理-导出字段（CSV）

## 说明
- 教师端「课程学生管理」页面支持导出 CSV。
- 导出字段用于后续在 Excel 里做筛选、透视表、分组统计。
- **雷达数据**来自后端聚合：五维分数、雷达面积（百分比）与总评级（A/B/C/D）。

## 导出文件（全量导出）
接口：`GET /api/teachers/analytics/course/{courseId}/students/export`

## 字段列表

### 基础信息
- `studentId`: 学生用户 ID
- `studentName`: 学生姓名（展示名）
- `studentNo`: 学号/标识

### 学习与课程行为
- `progress`: 课程进度（0-100）
- `averageGrade`: 课程相关作业平均分（0-100，可能为空）
- `activityLevel`: 活跃度（high/medium/low/inactive）
- `studyTimePerWeek`: 每周学习时长（小时）
- `completedLessons`: 已完成小节数
- `totalLessons`: 小节总数
- `lastActiveAt`: 最近活跃时间（ISO 字符串）

### 雷达概览
- `radarArea`: 雷达面积百分比（0-100）
- `radarClassification`: 总评级（A/B/C/D）

### 五维分数与分维评级
- 维度代码固定为：
  - `MORAL_COGNITION`
  - `LEARNING_ATTITUDE`
  - `LEARNING_ABILITY`
  - `LEARNING_METHOD`
  - `ACADEMIC_GRADE`

- 每个维度会导出两列：
  - `{CODE}_score`: 该维度分数（0-100，保留 1 位小数，可能为空）
  - `{CODE}_rating`: 该维度评级（A/B/C/D，可能为空）

#### 分维评级规则（当前版本）
- A: 分数 ≥ 85
- B: 分数 ≥ 70 且 < 85
- C: 分数 ≥ 60 且 < 70
- D: 分数 < 60

> 备注：该评级用于导出后快速分组统计；页面展示的 `radarClassification` 是另一套“总评级”逻辑（基于五维整体形态），两者含义不同。
